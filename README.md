# Big Red Media

A multi-page React portfolio concept for a creative media brand. The project explores strong visual identity, reusable content-driven components, and responsive presentation across project, pricing, testimonial, and contact journeys.

> Portfolio demo: Big Red Media is a fictional showcase project, not a registered business or client engagement.

## Features

- Routed home, portfolio, project detail, pricing, testimonial, and contact pages
- Reusable header, footer, and project-card components
- Project, pricing, and testimonial content separated into data modules
- Responsive CSS organised by page and feature
- Vite production workflow with ESLint checks

## Tech stack

React, React Router, Vite, JavaScript, and modular CSS.

## Run locally

```bash
git clone https://github.com/Dguminski12/big-red-media.git
cd big-red-media
npm install
npm run dev
```

Quality and production checks:

```bash
npm run lint
npm run build
```
